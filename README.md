# movie-app
full stack proj, (mongoose, express, authentication, authorization, angular)

## install
### server
```sh
cd <my_location>
npm install
```
### client
```sh
cd <my_location>/ClientApp
npm install
```

## run dev
### client
```sh
cd <my_location>/ClientApp
npm start
```
### server
```sh
cd <my_location>
npm run dev
```

## run prod
### client
```sh
cd <my_location>/ClientApp
npm run build
```

### server
```sh
cd <my_location>
npm start
```
